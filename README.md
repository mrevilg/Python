## Basic Concepts
| Operator Precedence | Explanations |
| ------------- | ------------- |
| **  | To the power of  |
| -X  | Negative number  |
| * / %  | Multiply, Divide, Modulus or Remainder |
| + -  | Add, Minus  |

input function will always return a string unless: int('*prompt*') * 7 etc to generate total, not string.
